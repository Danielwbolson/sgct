To build SGCT under windows please specify the SGCT_ROOT_DIR environment variable.

For Open Scene Graph please specify the folowing:

OSG_ROOT

And then:

OSGHOME = %OSG_ROOT%
OSG_FILE_PATH = %OSG_ROOT%\data
OSG_BIN_PATH = %OSG_ROOT%\bin

To path append:

%OSG_ROOT%\bin and %OSG_ROOT%\lib
